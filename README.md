# Sky Jump
오픈소스sw 최종 과제

### 플레이 방법 
스페이스 바를 눌러 진행하고 방향키로 플레이한다.

게임 방법 : 방향키를 통해 랜덤하게 움직이는 NPC(빨간색)을 피해 하늘색(PC)를 조작하며 핑크색(코인)을 획득한다. 코인을 모두 획득하면, 출구(핑크색)를 찾아 다음 스테이지로 이동한다.
